## 📖 关于项目

XivStrat 是一个专为最终幻想14玩家打造的攻略站点，起始于 Patch 7.2。我们致力于提供详细、清晰、准确、易懂的游戏攻略，帮助玩家更好地享受艾欧泽亚的冒险。

## ✨ 站点特色

- 🚀 基于 **Astro + Vue 3 + TailwindCSS + PixiJS + Framer Motion** 构建
- 🌙 支持 **明暗主题切换**
- 🎮 使用 **PixiJS** 提供可交互的攻略阅读体验
- ⚡ 未来还有更多功能正在开发中...
